## Trabajo realizado por Emilio Ortega Castro primer trimestre. ##

Esta base de datos tiene como finalidad hacer una coleccion sobre grupos musicales, y su principal cancion.
-La cancion está dentro de un array de tipo documento que contiene su nombre y duración.
-Los operadores utilizados son:
$ne: Indica Diferente
$and: El cual se usa cuando se cumple alguna de las condiciones
$or: Es usado cuando al menos una de las condiciones es cumplida
$eq: Indica igualdad
$in: Se usa para arrays e indica que contiene
$exists: Indica que un campo existe
